Introducing Double Number Line Diagrams

Let’s use number lines to represent equivalent ratios.

I can label a double number line diagram to represent batches of a recipe or color mixture. When I have a double number line that represents a situation, I can explain what it means.
double number line diagram:

A double number line diagram uses a pair of parallel number lines to represent equivalent ratios. The locations of the tick marks match on both number lines. In the double number line diagram shown here, an example of equivalent ratios is 3:5 and 6:10. The tick marks labeled with a 0 line up, but the other numbers are usually different.
