angle bisector

An angle bisector is a line through the vertex of an angle that divides it into two equal angles.

assertion

An assertion is a statement that one thinks is true but has not yet proved.

congruent

Two figures are congruent if there is a sequence of rigid motions (translations, rotations, and reflections) that take one figure onto the other.

conjecture

A conjecture is a reasonable guess that one is trying to either prove or disprove.

directed line segment

A directed line segment is a line segment with an arrow at one end specifying a direction.

inscribed

A polygon is inscribed in a circle if it fits inside the circle and every vertex of the polygon is on the circle. A circle is inscribed in a polygon if it fits inside the polygon and every side of the polygon is tangent to the circle.

line of symmetry

A line of symmetry for a figure is a line such that a reflection across it takes the figure onto itself.

The figure here shows two lines of symmetry for a regular hexagon, and two lines of symmetry for the capital letter I.

line segment

A line segment is a set of points on a line with two endpoints.

parallel

Two lines that don’t intersect are called parallel. Line segments are also parallel if they extend into parallel lines.

perpendicular bisector

The perpendicular bisector of a segment is a line through the midpoint of the segment that is perpendicular to it. 

reflection

A reflection is defined using a line. It takes one point to another point that is the same distance from the given line but on the other side. The segment from the original point to the image is perpendicular to the given line.

reflection symmetry

A figure has reflection symmetry if there is a reflection that takes the figure to itself.

regular polygon

A regular polygon is a polygon where all of the sides are congruent and all of the angles are congruent.

rigid transformation

A rigid transformation is a translation, rotation, or reflection. The term can also be used to refer to a sequence of these.

rotation

A rotation has a center and a directed angle. It takes one point on a circle to another using a given center. The two radii, the one from the center to the original point and the one from the center to the image, make the given angle.

rotation symmetry

A figure has rotation symmetry if there is a rotation that takes the figure onto itself. (We don’t count rotations using angles such as and that take every point on the figure back to its original position after the rotation.)

symmetry

A figure has symmetry if there is a rigid transformation which takes it onto itself (not counting a transformation that takes every point back to its original position).

tessellation

An arrangement of figures that covers the entire plane without gaps or overlaps.

theorem

A theorem is a statement that has been proved mathematically.

translation

A translation is defined using a directed line segment. A translation takes a point to another point so that the directed line segment from the original point to the image is parallel to the given directed line segment and has the same length and direction.
