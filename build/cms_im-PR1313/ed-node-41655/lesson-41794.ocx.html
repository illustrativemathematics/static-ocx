directed line segment:

A directed line segment is a line segment with an arrow at one end specifying a direction.

theorem:

A theorem is a statement that has been proved mathematically.

translation:

A translation is defined using a directed line segment. A translation takes a point to another point so that the directed line segment from the original point to the image is parallel to the given directed line segment and has the same length and direction.

A translation slides a figure in a given direction for a given distance with no rotation. The distance and direction is given by a directed line segment. The arrow of the directed line segment specifies the direction of the translation, and the length of the directed line segment specifies how far the figure gets translated.

More precisely, a translation of a point along a directed line segment is a transformation that takes to so that the directed line segment is parallel to , goes in the same direction as , and is the same length as .
